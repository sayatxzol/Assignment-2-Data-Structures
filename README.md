# Assignment 2: Data structures

## Project Overview
This project contains custom implementations of fundamental data structures in Java. [cite_start]The goal was to build logical data structures (Stack, Queue, MinHeap) using physical structures (ArrayList, LinkedList) without using the standard `java.util.*` collections[cite: 2, 3, 5].

## File Structure
* [cite_start]**MyList**: The base interface for all list-based structures[cite: 6, 7].
* [cite_start]**MyArrayList**: Dynamic array implementation.
* [cite_start]**Mylinkedlist**: Doubly linked list implementation.
* [cite_start]**Mystack**: LIFO structure built on a physical list[cite: 43, 45].
* [cite_start]**Myqueue**: FIFO structure built on a physical list[cite: 43, 45].
* [cite_start]**Myminheap**: Minimum heap implementation[cite: 43].
* [cite_start]**Main**: Test class to demonstrate and verify the functionality of all implemented methods.
